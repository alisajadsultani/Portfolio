Customer Churn Prediction

Analyzing and Predicting Customer Retention

About the Project

This project predicts customer churn using machine learning models like Decision Trees, Random Forest, and XGBoost. It identifies key factors such as tenure and monthly charges that contribute to customer churn.

Techniques like data encoding, oversampling with SMOTE, and hyperparameter tuning were applied to ensure high accuracy and recall.
