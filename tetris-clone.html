Tetris Clone

A fun and interactive game built with C++

About the Project

Developed using C++, this Tetris Clone demonstrates game development concepts like sprite animation, collision detection, and smooth 60 FPS gameplay. It features dynamic piece generation, rotation, and a scoring system with increasing difficulty.

Emulating classic Tetris, this project combines efficient programming and creative design for an engaging user experience.
