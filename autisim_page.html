Autism Detection

Advancing Early Detection Through Machine Learning

About the Project

This project harnesses the power of machine learning to aid in the early detection of Autism Spectrum Disorder (ASD). By analyzing behavioral and demographic data, it provides insights to assist healthcare professionals in making timely and accurate diagnoses.

Featuring models such as Decision Trees, Random Forests, and XGBoost, this initiative bridges the gap between data-driven insights and real-world applications, promoting better support for individuals with autism and their families.
